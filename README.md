## 🧠 About Me

I build **end-to-end AI systems** — from raw audio pipelines and autonomous agents to real-time vision analytics. My focus is on turning research-grade LLMs into reliable, scalable, deployable products.

- 🔗 **LangChain / LangGraph** — multi-step agentic workflows & tool orchestration
- 🎙️ **Voice Agents** — full pipeline: noise suppression → VAD → STT → LLM → TTS
- 📚 **RAG Systems** — retrieval-augmented generation with hybrid search & reranking
- 👁️ **Computer Vision** — real-time inference with DeepStream & VLM integration
- ⚡ **Local Inference** — Ollama + quantized models for on-device / edge deployment

## 🏛️ System Architectures

**🎙️ Full Voice Agent Pipeline**
```
Mic Input
  └─► [Noise Suppression]       ← RNNoise / DeepFilterNet / WebRTC-NS
        └─► [VAD]               ← Silero-VAD (speech boundary detection)
              └─► [STT]         ← faster-whisper
                    └─► [LangGraph Agent]
                          ├─► [Tool Calls / RAG / Memory]
                          └─► [LLM Response]
                                └─► [TTS]    ← ElevenLabs / Kokoro / Coqui
                                      └─► Audio Stream via LiveKit / WebSocket
```

**📚 RAG Pipeline**
```
Documents → [Chunking] → [Embedding Model] → [Vector Store]
Query     → [Retriever + Reranker] → [LLM + Context] → Answer
```

**👁️ DeepStream Vision Pipeline**
```
Camera / RTSP Stream
  └─► [DeepStream Pipeline (GStreamer)]
        ├─► [Primary Detector]              ← TensorRT YOLO
        ├─► [Multi-Object Tracker]          ← ByteTrack / DeepSORT
        ├─► [Secondary Classifier / VLM]    ← scene understanding
        └─► [Analytics + Alerts]            → API / Dashboard
```

**🤖 Multi-Agent System (LangGraph)**
```
Supervisor
  ├─► Research Agent  → web / RAG tools
  ├─► Analyst Agent   → code execution
  └─► Writer Agent    → final output
```

**⚡ Local Inference Stack**
```
Request → [FastAPI Gateway]
            ├─► Ollama        ← chat / instruction models (local)
            ├─► vLLM          ← high-throughput batched serving
            └─► TensorRT      ← vision / CV model serving (edge/GPU)
```
